# 🐾 Refugio de Mascotas - Stack Web con Docker

Este repositorio contiene la implementación del Laboratorio Práctico N°2 de la asignatura Taller de Ingeniería Informática. El proyecto consiste en una aplicación web dinámica (CRUD) para la gestión de un refugio de animales, desplegada sobre una arquitectura de contenedores aislados.

## 🚀 Tecnologías Utilizadas

- **Frontend / Backend:** PHP 8.2 (con Apache)
- **Base de Datos:** MySQL 8.0
- **Infraestructura y Orquestación:** Docker & Docker Compose

## 📋 Requisitos Previos

Para ejecutar este proyecto en tu entorno local, asegúrate de tener instalado:
- [Docker](https://docs.docker.com/get-docker/)
- [Docker Compose](https://docs.docker.com/compose/install/)
- Git
